  ## Executando

 1. Instale as dependências
    ```
    yarn
    #ou
    npm install
    ```
    1.2 (Opcional) Crie um arquivo `.env`na raiz do projeto para configurar algumas informações que o projeto irá usar
    ```
    PORT=8085
    ```

2. Execute o comando start
    ```
    yarn start
    #ou
    npm run start
    ```
